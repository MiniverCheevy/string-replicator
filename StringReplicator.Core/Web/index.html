Format String

Any .Net Format String should Work, plus a few extra:
{0:!} Friendly Name
{0:^} Proper Case
{#} 0 based row number
{+} 1 based row number

*be sure to double up the { and } characters in your format string
ex: public {0} {1} {{ get; set; }}
